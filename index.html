Sysdig Falco is an open source, host activity monitor powered by sysdig’s universal system level visibility. Write simple and powerful rules, and then output warnings in the format you need. Continuously monitor and detect container, application, host, or network activity... all in one place, from one source of data, with one syntax.

Sysdig was born from our own team's constant frustration. System level troubleshooting is just way more of a pain than it should be — especially in container-based environments. So we took the lessons we learned while helping build network monitoring tools like WinPCap and Wireshark and created a new kind of system troubleshooting tool for Linux.

See everything

Sysdig gives you instant access to a treasure trove of data buried within your system. Easily build sophisticated rules that see file access, system logs, network activity - your entire system - and inform you of every problem.

Designed for the rest of us

Too often, security tools and intrusion detection systems in are powerful but too complex for just anyone to use. Sysdig Falco is designed to make you productive in minutes.

Adapts to Your Environment

Augment the base rules to meet your security requirements. Flexible outputs mean you can pipe Sysdig Falco alerts to a broad collection of other tools and systems.

Container-friendly

Monitor containers from the outside, without any dependencies. Sysdig Falco’s rules are docker and rkt aware, meaning you can make container-specific rules. Falco understands the ecosystem as well - Kubernetes, Mesos, and Swarm metadata can be leveraged in your monitoring rules. and the risk of crashes.

Back-test rules

Create rules and then test them on historical system capture files. Know exactly what you’re going to get, before you go into production!

Fast and Stable Core

We leverage the sysdig kernel probe which has been in use for years, across hundreds of thousands of hosts.
